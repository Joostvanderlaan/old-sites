== Description ==

GPP Category Slider

This plugin creates a simple category slider on the apps hook of the base and its child themes.


== Installation ==

1. Upload the whole `gpp-category-slider` directory to the `/wp-content/plugins/` directory
1. Activate the 'GPP Category Slider' plugin through the 'Plugins' menu in WordPress
